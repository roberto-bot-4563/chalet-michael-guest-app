# Chalet Michael Version 2

Modern multilingual Next.js guest app.

## Publish

1. Delete the old files in the GitHub repository.
2. Upload all files and folders from this package to the repository root.
3. Commit to `main`.
4. Vercel will redeploy automatically.

Before publishing, replace `+41000000000` in `app/page.js` with Petra's phone number.
